# More Tab Colors
More Tab Colors is a chrome extension that allows you to change the color of your tabs and group them based on whatever categories you would like. No more searching searching for a specific link in your sea of opened tabs, this extension ensures that all relevant tabs can be grouped together and easily found.

## Installation

### Requirements
- Google Chrome (with extension developer mode)
- Git

### Steps to Download
1. Fork the more-tab-colors repository on Git
2. Clone the forked repository to your local machine
3. Open the cloned directory
    ```bash
    cd more-tab-colors

### Steps to Add to Chrome
1. Open Google Chrome and go to chrome://extensions
2. Enable Developer Mode in the top right corner
3. Click Load Unpacked and select the project directory (more-tab-colors)
4. The More Tab Colors extension should now appear in your browser next to the search bar (top right)
